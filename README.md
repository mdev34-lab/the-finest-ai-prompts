# The Finest (-ish) AI Skills

A curated collection of LLM and AI agent directives for real work and usage that doesn't *suck*.
Each directive follows the OpenCode Agent Skills format: `agent-skills/<name>/SKILL.md`.

## 🚀 Available Skills

### Core
- **[alpha](agent-skills/alpha/SKILL.md)** — Generalist system directive combining traits from all other skills
- **[coder](agent-skills/coder/SKILL.md)** — Production-grade software development with rigorous testing and validation
- **[deepcoder](agent-skills/deepcoder/SKILL.md)** — Exhaustive technical analysis for complex implementations
- **[sherlock](agent-skills/sherlock/SKILL.md)** — Deep research with cross-source verification and hypothesis testing

### Specialized
- **[lifewise](agent-skills/lifewise/SKILL.md)** — Christ-centered life guidance and spiritual formation
- **[r-prompt-legacy](agent-skills/r-prompt-legacy/SKILL.md)** — Forced chain-of-thought reasoning for older LLMs
- **[r-image](agent-skills/r-image/SKILL.md)** — High-quality image generation through structured intent analysis
- **[image-to-prompt](agent-skills/image-to-prompt/SKILL.md)** — Converts images into detailed text descriptions

### Designer
- **[designer-xl](agent-skills/designer-xl/SKILL.md)** — Extended UI/UX design with deep archetype library
- **[designer](agent-skills/designer/SKILL.md)** — Full UI/UX design directive with narrative and atmosphere
- **[designer-mini](agent-skills/designer-mini/SKILL.md)** — Compact version for smaller context windows
